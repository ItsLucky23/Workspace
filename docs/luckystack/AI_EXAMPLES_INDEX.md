| `trycatch` | The tryCatch [error, result] pattern | Whenever a call can throw — JSON parsing, DB/IO, an external fetch — wrap it in the LuckyStack `tryCatch` so failures come back as a checked `[error, result]` tuple instead of an unwound stack.
