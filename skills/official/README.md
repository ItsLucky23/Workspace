# Official Skills

Drop official Anthropic Claude skills here. Managed by Anthropic; the framework team does not touch this folder.

This folder is intentionally empty for now. When official skills are added (either by `npm install` of a skill bundle or by manual drop), they live under `skills/official/<skill-name>/` with their own `SKILL.md`.

For framework- and project-specific skills, see `../custom/`.
